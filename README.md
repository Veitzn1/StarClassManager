# 学校的班级常规管理

![image](./%E4%B8%BB%E9%A1%B5.png)
## 一、项目概述
本项目是一个班级常规数字化管理系统，旨在实现学校班级扣分管理的数字化、自动化。系统提供了登录、注册、数据导入、分数更正、班级量化展示等功能，方便学校管理人员对班级的各项指标进行管理和监控。


## 二、功能特性
1. **用户管理**：支持用户登录和注销功能，可记住用户名和密码，提高使用便捷性。
2. **数据管理**：
    - 支持数据导入，将班级扣分数据保存到数据库中。
    - 提供分数更正功能，可对已有的分数进行修改。
3. **数据展示**：
    - 提供排行榜，展示班级的排名情况。
    - 提供班级量化蛛网图，直观展示班级各项指标的得分情况。
    - 支持按年度和周次筛选数据。
4. **数据导出**：支持将班级量化蛛网图导出为图片，也可将所有图表打包下载为 ZIP 文件。

### 文件说明
1. **.user.ini**：配置 PHP 的 `open_basedir`，限制文件访问范围。
2. **correction-page.php**：分数更正页面，提供年度、周次、年级、班级、项目等选择，可输入新分值进行更正。
3. **class_radar_chart.php**：班级量化蛛网图页面，可选择年度，展示班级各项指标的得分情况，并支持图表导出。
4. **header.php**：页面头部，包含系统标题、导航栏和登录模态框。
5. **index.php**：主页，提供年度选择功能，班级评分数据录入功能。
6. **register.php**：用户注册处理文件。
7. **registration-frontend.php**：用户注册前端页面。
8. **save_data.php**：数据保存文件，处理数据导入和保存到数据库的操作。
9. **style.css**：全局样式文件，定义了页面的整体风格和布局。
10. **update_score.php**：分数更新文件，处理分数更正的请求。
11. **getinfo.php**：数据获取文件，提供获取年份列表、按年度获取扣分数据、获取项目分数等接口。

## 四、运行环境
1. **Web 服务器**：宝塔等
2. **PHP 版本**：建议使用 PHP 7.0 及以上版本。
3. **数据库**：MySQL 数据库。


## 六、使用说明
1. **登录与注册**：访问系统主页，点击“登录”按钮，输入用户名和密码进行登录。若没有账号，可点击“注册”按钮进行注册。
2. **数据导入**：在相应页面选择要导入的数据文件，点击“导入”按钮，将数据保存到数据库中。
3. **分数更正**：点击导航栏中的“更正”按钮，进入分数更正页面。选择年度、周次、年级、班级、项目，输入新分值，点击“提交更正”按钮。
4. **班级量化展示**：点击导航栏中的“班级量化”按钮，进入班级量化蛛网图页面。选择年度，系统将展示该年度各班的量化指标得分情况。可点击“导出图片”按钮导出单个图表，或点击“下载所有图片”按钮将所有图表打包下载为 ZIP 文件。

## 七、注意事项
1. **CSRF 防护**：系统使用 CSRF 令牌进行防护，确保请求的合法性。在进行表单提交时，请确保 CSRF 令牌正确传递。
2. **数据验证**：在输入数据时，请确保数据符合系统要求，如分数范围在 0 - 10 之间。
